Learn About UNC Charlotte's Orchestra

Project Overview — Prepared by Saige Hatfield

Overview ~ Client ~ Site Map ~ Page Design ~ Dynamic Functionality ~ Notes

Application & Purpose

The website will modernize and centralize all orchestra information — concerts, member bios, media, and joining details — in one elegant, accessible hub. It improves upon the existing COAA Ensembles page, which mixes orchestra content with other ensembles.

Intended Users

  • UNC Charlotte students (prospective/current music majors)
  • Orchestra members & faculty
  • Parents, alumni, and donors
  • Local concertgoers

Planned Content

  • Home — Mission & featured next concert
  • Events — Upcoming & past concerts
  • About — History, conductor, concertmaster, members
  • Gallery — Photos & videos
  • Join / Contact — Inquiry form & joining info

Page Design Summary

Home

  • Purpose: Introduce the orchestra, highlight next concert.
  • Content: Hero image, mission, concert preview, quick links.
  • Future JS: Smooth scrolling, event modal.

Events

  • Purpose: List upcoming and past concerts.
  • Content: Date, venue, description, Google Maps link.
  • Future JS: Filter by date, load event details from JSON.

About

  • Purpose: Share orchestra history, bios, and mission.
  • Future JS: jQuery UI dialog for bios.

Gallery

  • Purpose: Showcase photos and videos of performances.
  • Future JS: Lightbox or slideshow functionality.

Join / Contact

  • Purpose: Allow visitors to reach out or join.
  • Future JS: Client-side validation and mailto fallback.

Planned Dynamic Features

  • Event filtering with jQuery
  • Lightbox gallery plugin
  • Modal pop-ups for bios
  • Form validation for contact page
  • AJAX loading of event data (from data/events.json)

These will be added after static layout completion.
